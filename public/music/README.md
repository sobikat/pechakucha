# Background music

Add a coffee-shop / jazzy track here so the **Play music** button works.

1. **Download** a royalty-free track (e.g. from [Pixabay Music](https://pixabay.com/music/search/coffee%20bar/) or [Mixkit](https://mixkit.co/free-stock-music/lounge/)).
2. **Rename** the file to `coffee-shop.mp3`.
3. **Place** it in this folder: `public/music/coffee-shop.mp3`.

The page will loop this track when the user clicks the music button (♪) in the bottom-right area.
